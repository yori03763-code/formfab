# FormFab

**Describe it. Print it. Ship it.**

Text/Image to 3D-printed product service.

## Quick Start

```bash
# Frontend
cd frontend && npm install && npm run dev

# Backend
cd backend && npm install && npm run dev
```

## Tech Stack

- Frontend: Next.js 14 + Three.js + TailwindCSS
- Backend: Fastify + PostgreSQL
- APIs: Meshy AI + Shapeways
